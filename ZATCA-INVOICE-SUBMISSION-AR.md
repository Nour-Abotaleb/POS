# دليل إرسال الفواتير الضريبية لهيئة الزكاة (ZATCA)

## 📋 الفرق بين نوعين الفواتير

---

## 1️⃣ الفاتورة الضريبية المبسطة (B2C) - للمطاعم ✅

### 🎯 هذا النوع ينطبق على مطعمك!

### كيف يتم إرسال الفواتير؟

**الإجابة المختصرة: لا يتم إرسالها تلقائياً! ❌**

### ✅ ما هو مطلوب منك:

#### المرحلة الأولى: التسجيل والموافقة (مرة واحدة فقط)
```
1. التسجيل في بوابة فاتورة
   └─ https://fatoora.zatca.gov.sa

2. رفع نماذج الفواتير (PDF)
   ├─ فاتورة ضريبية مبسطة
   ├─ إشعار مدين (إذا كان موجود)
   └─ إشعار دائن (إذا كان موجود)

3. انتظار موافقة الهيئة
   └─ عادة تستغرق 1-3 أيام عمل

4. الحصول على شهادة الموافقة
   └─ تستطيع البدء في إصدار الفواتير
```

#### المرحلة الثانية: الاستخدام اليومي (بعد الموافقة)
```
✅ تصدر الفواتير بشكل عادي من النظام
✅ كل فاتورة تحتوي على QR Code
✅ العميل يمسح QR Code للتحقق (اختياري)
✅ لا يوجد إرسال تلقائي للهيئة
✅ الهيئة تراقب عشوائياً
```

### 📊 كيف تراقب الهيئة الفواتير؟

```
1. المراقبة العشوائية:
   └─ الهيئة قد تطلب عينات من الفواتير في أي وقت

2. التفتيش الميداني:
   └─ المفتشون قد يزورون المطعم ويفحصون الفواتير

3. شكاوى العملاء:
   └─ إذا اشتكى عميل، الهيئة تطلب الفاتورة

4. التقارير الدورية:
   └─ تقدم تقرير ضريبي شهري/ربع سنوي
   └─ يحتوي على إجمالي المبيعات والضرائب
   └─ لا يحتوي على تفاصيل كل فاتورة
```

### ✅ ما يجب عليك فعله:

```
1. حفظ نسخة من كل فاتورة:
   ├─ في النظام (قاعدة البيانات)
   ├─ نسخة احتياطية (Backup)
   └─ لمدة 6 سنوات على الأقل

2. التأكد من صحة QR Code:
   └─ كل فاتورة يجب أن تحتوي على QR Code صحيح

3. الاحتفاظ بسجلات منظمة:
   └─ لتقديمها عند الطلب

4. تقديم التقارير الضريبية:
   └─ شهرياً أو ربع سنوياً حسب حجم المبيعات
```

## 2️⃣ الفاتورة الضريبية الكاملة (B2B) - للشركات ❌

### 🎯 هذا النوع لا ينطبق على المطاعم!

### كيف يتم إرسال الفواتير؟

**الإجابة: يتم إرسالها تلقائياً عبر API ✅**

### ⚠️ متطلبات B2B (غير مطلوبة للمطاعم):

```
1. API Integration مع ZATCA
   └─ كل فاتورة تُرسل فوراً للهيئة

2. Cryptographic Stamp
   └─ توقيع رقمي لكل فاتورة

3. Digital Signature
   └─ شهادة رقمية من جهة معتمدة

4. Invoice Hash
   └─ كل فاتورة لها Hash فريد

5. Real-time Reporting
   └─ إرسال فوري خلال 24 ساعة
```

---

## 🎯 ملخص: ما هو مطلوب من مطعمك؟

### ✅ المطلوب (B2C):

```
1. التسجيل في بوابة فاتورة (مرة واحدة)
2. رفع نماذج الفواتير للموافقة (مرة واحدة)
3. إصدار فواتير تحتوي على:
   ├─ QR Code صحيح
   ├─ رقم VAT
   ├─ السجل التجاري
   ├─ جميع البيانات المطلوبة
   └─ ضريبة 15%

4. حفظ نسخ من الفواتير (6 سنوات)
5. تقديم التقارير الضريبية الدورية
6. تقديم الفواتير عند الطلب
```

### ❌ غير المطلوب (B2B):

```
❌ لا API Integration
❌ لا إرسال تلقائي للفواتير
❌ لا Cryptographic Stamp
❌ لا Digital Signature
❌ لا Real-time Reporting
```

## 📝 خطوات التسجيل في بوابة فاتورة

### الخطوة 1: إنشاء حساب
```
1. اذهب إلى: https://fatoora.zatca.gov.sa
2. اضغط على "تسجيل جديد"
3. أدخل بيانات المطعم:
   ├─ رقم السجل التجاري
   ├─ رقم VAT
   ├─ بيانات المسؤول
   └─ رقم الجوال للتفعيل
4. فعّل الحساب عبر SMS
```

### الخطوة 2: رفع نماذج الفواتير
```
1. سجّل دخول على البوابة
2. اذهب إلى "رفع نماذج الفواتير"
3. ارفع ملفات PDF:
   ├─ فاتورة ضريبية مبسطة (إلزامي)
   ├─ إشعار مدين (إذا كان موجود)
   └─ إشعار دائن (إذا كان موجود)
4. اضغط "إرسال للمراجعة"
```

### الخطوة 3: انتظار الموافقة
```
⏱️ المدة: 1-3 أيام عمل

✅ إذا تمت الموافقة:
   └─ تستلم شهادة موافقة
   └─ تستطيع البدء في إصدار الفواتير

❌ إذا تم الرفض:
   └─ تستلم ملاحظات التعديل
   └─ تعدّل الفواتير وتعيد الرفع
```

### الخطوة 4: البدء في الاستخدام
```
✅ بعد الموافقة:
   ├─ أصدر الفواتير من النظام بشكل عادي
   ├─ كل فاتورة تحتوي على QR Code
   ├─ احفظ نسخة من كل فاتورة
   └─ قدّم التقارير الضريبية الدورية
```

---

## 📊 التقارير الضريبية الدورية

### ما هي التقارير المطلوبة؟

```
1. الإقرار الضريبي (شهري أو ربع سنوي):
   ├─ إجمالي المبيعات
   ├─ إجمالي الضريبة المحصّلة
   ├─ إجمالي المشتريات
   ├─ إجمالي الضريبة المدفوعة
   └─ صافي الضريبة المستحقة

2. لا يحتوي على:
   ❌ تفاصيل كل فاتورة
   ❌ أسماء العملاء
   ❌ تفاصيل الأصناف
```

### كيف تُقدّم التقارير؟

```
1. عبر بوابة الهيئة:
   └─ https://zatca.gov.sa

2. أو عبر محاسب قانوني:
   └─ يقدم التقارير نيابة عنك

3. المواعيد:
   ├─ شهرياً: إذا المبيعات > 40 مليون ريال سنوياً
   └─ ربع سنوياً: إذا المبيعات < 40 مليون ريال سنوياً
```

## 🔍 ماذا يحدث عند التفتيش؟

### سيناريو 1: تفتيش ميداني
```
1. المفتش يزور المطعم
2. يطلب فواتير عشوائية
3. يمسح QR Code للتحقق
4. يتأكد من:
   ├─ صحة البيانات
   ├─ وجود رقم VAT
   ├─ صحة الضريبة 15%
   └─ وجود السجل التجاري
```

### سيناريو 2: طلب فواتير محددة
```
1. الهيئة ترسل طلب رسمي
2. تحدد أرقام الفواتير المطلوبة
3. لديك 5 أيام عمل للرد
4. ترسل نسخ PDF من الفواتير
```

### سيناريو 3: شكوى عميل
```
1. عميل يشتكي من فاتورة
2. الهيئة تطلب نسخة من الفاتورة
3. تتحقق من صحة البيانات
4. تصدر قرارها
```

---

## ⚠️ العقوبات المحتملة

### إذا لم تلتزم بالمتطلبات:

```
1. عدم إصدار فاتورة:
   └─ غرامة: 5,000 - 50,000 ريال

2. فاتورة غير متوافقة:
   └─ غرامة: 5,000 ريال لكل فاتورة

3. عدم حفظ السجلات:
   └─ غرامة: 10,000 - 50,000 ريال

4. عدم تقديم التقارير:
   └─ غرامة: 5% - 25% من الضريبة المستحقة
```

## ✅ نصائح للالتزام الكامل

### 1. النظام الحالي (جاهز):
```
✅ يولّد QR Code صحيح
✅ يعرض جميع البيانات المطلوبة
✅ يحفظ الفواتير في قاعدة البيانات
✅ يطبع فواتير متوافقة
```

### 2. ما يجب عليك فعله:
```
1. خذ Backup يومي من قاعدة البيانات
2. احفظ نسخ PDF من الفواتير المهمة
3. راجع الفواتير دورياً للتأكد من صحتها
4. درّب الموظفين على إصدار الفواتير الصحيحة
5. قدّم التقارير الضريبية في موعدها
```

### 3. أدوات مساعدة:
```
1. تطبيق ZATCA:
   └─ للتحقق من QR Code

2. بوابة فاتورة:
   └─ لرفع النماذج والتقارير

3. نظام POS الحالي:
   └─ يولّد الفواتير تلقائياً
```
